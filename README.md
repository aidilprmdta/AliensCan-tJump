# 👽 Aliens Can't Jump

> A simple and fun 2D game where aliens must survive without the ability to jump.

![Preview](https://github.com/aidilprmdta/AliensCan-tJump/blob/main/preview.png?raw=true)

## 🎮 Description

**Aliens Can't Jump** is a casual 2D game where players control an alien character that cannot jump.  
The challenge is to survive, avoid obstacles, and strategize movement using limited mechanics.

This project was built to explore:
- Game logic implementation
- Object movement & collision
- Basic game design principles

---

## ✨ Features

- 🚫 Unique mechanic: **No jumping**
- 🎯 Obstacle avoidance gameplay
- ⚡ Simple and lightweight
- 🎮 Easy to play, hard to master

---

## 🛠️ Tech Stack

- Language: `Alpine.js`
- Framework: `Tailwindcss`
- Tools: Git, GitHub

## ⚙️ Installation & Run

1. Clone repository
```bash

````

2. Masuk ke folder

```bash
cd AliensCan-tJump
```

3. Jalankan project

* Open `index.html` di browser
  atau gunakan live server
